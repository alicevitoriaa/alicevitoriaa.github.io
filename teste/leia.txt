Olá, meu nome é Alice Vitória e eu vou explicar a seguir como essa interface foi desenvolvida...

HTML:

Onde: Utilizado em todo o documento (<!DOCTYPE html> até </html>).
Porquê: O HTML é a estrutura básica do site, fornecendo a organização e o conteúdo semântico.

<!---->

CSS:

Onde: Referenciado nos links do head (<link rel="stylesheet" type="text/css" href="styles/style.css">).
Porquê: O CSS é usado para estilizar e dar design ao seu site, tornando-o atraente e coeso visualmente.

<!---->

Bootstrap:

Onde: Referenciado no link do head (<script src="js/bootstrap.min.js"></script>).
Porquê: O Bootstrap contribuiu para o responsivo do site, economizando código e tempo.

<!---->

SweetAlert:

Onde: Referenciado no link do head (<script src="https://cdn.jsdelivr.net/npm/sweetalert2@11"></script>).
Porquê: O SweetAlert é utilizado para criar modais de forma mais estilizada e atraente, oferecendo uma experiência de usuário mais agradável, especialmente em dispositivos responsivos.

<!---->

JavaScript e JQUery (js/Jquery.js e js/script.js):

Onde: Referenciado nos scripts no final do body (<script src="js/Jquery.js"></script>, <script src="js/script.js"></script>).
Porquê: O JavaScript foi usado para o carregamento do array de estados e cidades a partir do estado escolhido pelo usuário. Já o jQuery simplificou tarefas, como envio de dados para API e ate mesmo a formatação de número de celular no formulário.
